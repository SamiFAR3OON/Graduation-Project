# FitLife – Graduation Project Landing Page

## Project Overview
FitLife is a graduation project landing page designed to introduce a fitness mobile application that helps users achieve their health and fitness goals through personalized workout plans, progress tracking, and data-driven recommendations.

The landing page was built using HubSpot Design Manager and focuses on clarity, usability, and full content editability through the CMS. The project demonstrates the ability to create reusable, flexible, and well-structured custom modules using HTML, CSS, JavaScript, and HubL.

The website presents the project idea, core features (MVP), team members, and contact information in a professional and modern layout suitable for real-world use.

---

## Modules Used
The landing page consists of **4 custom modules**, each built to be fully editable and reusable:

1. **Hero Section Module**
   - Project title
   - Short project description
   - Call-to-action button
   - Hero image
   - Animated text using JavaScript

2. **Features / MVP Module**
   - Displays the main features of the FitLife application
   - Uses a flexible card-based layout
   - Editable icons, titles, and descriptions for each feature

3. **Team Section Module**
   - Introduces the project team members
   - Each member includes:
     - Profile image
     - Name
     - Role
     - Detailed description of responsibilities
   - Designed with image on the right and text on the left for better readability

4. **Contact Section Module**
   - Contact title and description
   - Call-to-action button with editable link
   - Used to encourage users or stakeholders to reach out

---

## Technologies Used
- **HTML5** – Structure and semantic layout
- **CSS3** – Styling, animations, and responsive design
- **JavaScript** – Simple animations and interactions
- **HubL** – Dynamic fields, conditions, and loops for CMS editability
- **HubSpot Design Manager** – Module creation and drag-and-drop page building
